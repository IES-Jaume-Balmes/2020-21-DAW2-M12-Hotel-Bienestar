Description

Este proyecto es el burdo intento de hacer una pagina web de un hotel (Hotel Bienestar) con el uso de frameworks de frontend como el react y de backend como el express y el sequelize.

Documentation

Documentacion Hotel Bienestar Como iniciar el proyecto Utilizad dos terminales, el primero nos servirá para tener activo el cliente y el segundo para el servidor. Iniciar el cliente Vamos a la carpeta del cliente y ejecutamos el siguiente comandos para instalar las dependencias: npm install e inciamos el cliente en el puerto 3000 npm start Iniciar el Servidor Ahora iniciaremos el servidor, pero antes tendremos que tener la base de datos lista, en nuestro caso utilizamos el xampp con phpmyadmin, dicho esto creamos una base de datos con el nombre de “hotelbd2”.
